## Update Major Version

### 14.4

```sh
freebsd-update upgrade -r 14.4-RELEASE
```


### 15.0

```sh
freebsd-update upgrade -r 15.0-RELEASE
```

## Install & Update

```sh
freebsd-update fetch
freebsd-update install
reboot
```

### Check Version

```sh
freebsd-version
```
